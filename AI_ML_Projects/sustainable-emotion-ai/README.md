# 🧠 Sustainable Emotion AI

A full-stack artificial intelligence application designed to recognize human emotions from images/video while emphasizing sustainable computing practices and resource efficiency.

## 🌟 Features
- **Real-time Emotion Recognition**: Detects multiple emotions with high accuracy.
- **Deep Learning Backend**: Powered by a CNN-based architecture optimized for performance.
- **Modern Frontend Interface**: High-fidelity UI for seamless user interaction and visualization.
- **Sustainable Architecture**: Optimized resource usage for reduced computational footprint.

## 🏗️ Project Architecture

The system is split into two primary components:
1.  **[Backend](./backend)**: Python-based FastAPI server handling model inference and data processing.
2.  **[Frontend](./frontend)**: Interactive UI for capturing input and displaying emotion diagnostics.

---

## 🚀 Execution Guide

### 1. Backend Setup
1. Navigate to the `backend/` directory.
2. Install dependencies: `pip install -r requirements.txt`.
3. Start the server: `./backend_start.sh`.

### 2. Frontend Setup
1. Open the `frontend/` directory.
2. Follow the specific instructions in the frontend directory for launching the UI.
