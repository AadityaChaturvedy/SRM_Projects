# 🧠 Sustainable Emotion AI - Backend

The core processing engine for the Sustainable Emotion AI project, utilizing deep learning to analyze human emotions from image and video data.

## 🛠️ Tech Stack
- **Runtime**: Python 3.10+
- **Framework**: FastAPI / Flask
- **Machine Learning**: TensorFlow / PyTorch for CNN inference.
- **Key Features**: High-performance REST API for real-time emotion diagnostics.

## 🚀 Execution Guide
1.  **Environment Setup**:
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows: venv\Scripts\activate
    pip install -r requirements.txt
    ```
2.  **Start Server**:
    -   **Unix/macOS**: `./backend_start.sh`
    -   **Windows**: `backend_start.bat`
