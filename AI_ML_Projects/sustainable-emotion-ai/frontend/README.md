# 🌐 Sustainable Emotion AI - Frontend

The interactive user interface for the Sustainable Emotion AI project, designed for seamless real-time emotion visualization.

## 🛠️ Tech Stack
- **Framework**: Modern JavaScript (ES6+) / React
- **Visualization**: D3.js / Canvas for emotion tracking charts.
- **Real-Time Feed**: Camera integration via WebRTC/Browser APIs.

## 🚀 Setup
- Open the frontend files in a modern web browser to interact with the system's capture and analysis tools.
- Ensure the **[Backend](../backend)** is running to receive live emotion diagnostics.
