# 🗄️ EduBase

A specialized repository management system for educational institutions, focusing on secure student and staff records.

## 🌟 Features
- **Centralized Data Management**: Secure storage and retrieval of academic information.
- **Role-Based Access**: Permission-driven management for faculty and students.
- **Reporting Utilities**: Generate transcripts and departmental records efficiently.

## 🛠️ Tech Stack
- **Framework**: Java (Swing/JDBC)
- **Database**: SQLite (`edubase.db`)
- **Integration**: Works in tandem with **[EduTrack](../EduTrack)** for end-to-end management.
