# 📚 EduTrack & EduBase

A pair of integrated educational management systems for tracking student performance and maintaining a central academic database.

## 🌟 Features
- **EduTrack**: Real-time performance monitoring, grade tracking, and attendance.
- **EduBase**: Robust backend for storing student records, faculty information, and curricula.
- **Seamless Integration**: Automated data synchronization between both modules.

## 🛠️ Tech Stack
- **Framework**: Java Swing / JDBC
- **Database**: SQLite (`edutrack.db`, `edubase.db`)
- **Project Structure**: Multi-module Maven setup.
