# ⚡ Electricity Billing System

A modern utility management application for calculating, billing, and tracking electricity consumption for residential and commercial customers.

## 🌟 Features
- **Dynamic Tariff Calculation**: Automated billing based on tiered usage models.
- **Customer Dashboard**: Historical usage patterns and payment tracking.
- **Payment Processing**: Simulated online payment integration and receipt generation.
- **Administrative Control**: Manage meter readings and customer records.

## 🛠️ Tech Stack
- **Framework**: Java Desktop
- **Data Store**: JSON-based (`data.json`) for lightweight persistence.
- **Build**: Maven
