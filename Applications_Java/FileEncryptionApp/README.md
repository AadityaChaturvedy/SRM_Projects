# 🔐 File Encryption Utility

A security-focused tool for performing high-performance file encryption and decryption, ensuring data privacy and secure transmission.

## 🌟 Features
- **AES-256 Encryption**: Industry-standard cryptographic algorithms.
- **Batch Processing**: Secure multiple files or entire directories simultaneously.
- **User-Friendly Interface**: Intuitive desktop UI for easy key management.
- **Zero-Trust Architecture**: Local encryption keys with no cloud dependency.

## 🛠️ Tech Stack
- **Language**: Java 17
- **Security**: Java Cryptography Extension (JCE)
- **Framework**: Java Swing / JavaFX
- **Build**: Maven
