# 🎓 Student Database System

A performant, lightweight database application for managing student records, course enrollments, and academic performance.

## 🌟 Features
- **Scalable Record Management**: Handle thousands of student records with low latency.
- **Enrollment Tracking**: Automated course capacity management and waitlisting.
- **Grade Analytics**: Aggregate and individual performance reports.

## 🛠️ Tech Stack
- **Framework**: Java
- **Database**: SQLite (`students.db`)
- **Scripts**: Includes `build.sh` for automated deployment.
