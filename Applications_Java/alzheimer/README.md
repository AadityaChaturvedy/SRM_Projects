# 🧬 Alzheimer's Patient Management

A specialized healthcare application designed to assist caregivers and medical professionals in tracking patient health metrics, medication schedules, and cognitive assessments.

## 🌟 Features
- **Patient Profiles**: Comprehensive history and medication tracking.
- **Cognitive Assessments**: Logging and trend analysis of cognitive test results.
- **Caregiver Dashboard**: Critical alerts and daily task management.
- **Data Privacy**: Secure local storage of sensitive medical information.

## 🛠️ Tech Stack
- **Framework**: Java Desktop (Swing)
- **Database**: SQLite (`alzheimer_app.db`)
- **Architecture**: DAO Pattern for persistence logic.
