# 💰 Cash Recorder Final

A modern financial ledger application for tracking personal or small business transactions with high precision and reporting capabilities.

## 🌟 Features
- **Transaction Logging**: Fast and accurate entry of income and expenses.
- **Category Management**: Customizable transaction tagging for detailed analysis.
- **Financial Reporting**: Visual summaries of spending patterns and budget adherence.
- **Data Export**: Support for exporting records to CSV/Excel formats.

## 🛠️ Tech Stack
- **Framework**: Java Desktop
- **Database**: SQLite (`cash_recorder.db`)
- **Build**: Maven wrapper (`mvnw`) included.
