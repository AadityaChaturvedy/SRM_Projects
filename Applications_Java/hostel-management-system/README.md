# 🏩 Hostel Management System

A desktop application designed to streamline student registration, room allocation, and financial records for university hostels.

## 🌟 Features
- **Student Management**: Register, update, and manage student details.
- **Room Allocation**: Automated and manual room assignment logic.
- **Financial Ledger**: Track fee payments, outstanding balances, and receipt generation.
- **Dashboard Analytics**: Quick view of occupancy and revenue metrics.

## 🛠️ Technical Implementation
- **Framework**: Java Swing
- **Database**: SQLite (`hostel.db`)
- **Persistence Layer**: Custom DAO patterns.
- **Build Tool**: Maven

## 🚀 Execution Guide

1.  **Prerequisites**: Ensure JDK 11+ is installed.
2.  **Build Project**:
    ```bash
    mvn clean package
    ```
3.  **Run Application**:
    ```bash
    java -jar target/hostel-management-system-1.0-SNAPSHOT.jar
    ```
    *Alternatively, use `run-hostel-app.bat` on Windows.*
