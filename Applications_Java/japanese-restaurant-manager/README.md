# 🍱 Japanese Restaurant Manager

A comprehensive Point-of-Sale (POS) and inventory management system designed for the unique operational requirements of a Japanese restaurant.

## 🌟 Features
- **Dynamic Menu Management**: Easy updates for seasonal dishes and pricing.
- **Table Reservation System**: Track bookings and real-time table status.
- **Inventory Tracking**: Monitor ingredients and automated low-stock alerts.
- **Billing & Analytics**: Detailed sales reporting and receipt generation.

## 🛠️ Tech Stack
- **Framework**: JavaFX 17+
- **Database**: SQLite (`restaurant.db`)
- **Integration**: Maven-based project structure.

## 🚀 Execution Guide
1.  **Build**: `mvn clean package`
2.  **Run**: `java -jar target/japanese-restaurant-manager-1.0-SNAPSHOT.jar`
