# 🖥️ Planner Backend

The server-side implementation for the Full-Stack Academic Planner, handling API requests and data persistence.

## 🛠️ Tech Stack
- **Runtime**: Node.js / Python
- **Database**: JSON-based storage (`timetable.json`)
- **Key Features**: RESTful API endpoints, CRUD operations for schedules.

## 🚀 Setup
1.  Install dependencies: `npm install` or `pip install -r requirements.txt`
2.  Start the server: `node index.js` or `python main.py`
