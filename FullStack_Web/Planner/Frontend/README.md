# 🌐 Planner Frontend

The client-side interface for the Full-Stack Academic Planner, built for high responsiveness and a rich user experience.

## 🛠️ Tech Stack
- **Framework**: React.js / JavaScript (ES6+)
- **Styling**: Vanilla CSS / Responsive Design
- **Integration**: Communicates with the Planner Backend for real-time schedule updates.

## 🚀 Setup
1.  Navigate to the directory and install dependencies: `npm install`
2.  Start the development server: `npm start`
