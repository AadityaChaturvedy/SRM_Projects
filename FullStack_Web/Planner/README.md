# 📅 Full-Stack Academic Planner

A modern web application for managing academic schedules, timetables, and task lists, featuring a clean separation of concerns between its frontend and backend layers.

## 🌟 Features
- **Interactive Timetable**: Drag-and-drop or grid-based schedule management.
- **Dynamic Data Storage**: Real-time updates and persistence.
- **Task Management**: Integrated to-do lists linked to specific academic events.
- **User-Centric UI**: Responsive design for both desktop and mobile use.

## 🏗️ Project Architecture

The application is structured into two main components:
1.  **[Backend](./Backend)**: Node.js/Python API for handling data persistence and scheduling logic.
2.  **[Frontend](./Frontend)**: Modern React/JavaScript interface for a rich user experience.
3.  **Data Source**: Centralized `timetable.json` for lightweight state management.

## 🚀 Execution Guide

### Prerequisites
- Node.js (v16+) or Python (v3.10+)
- npm or pip

### Setup
Refer to the individual `README.md` files in the **[Backend](./Backend)** and **[Frontend](./Frontend)** directories for specific environment setup and execution commands.
