# 📟 IoT & Embedded Systems Portfolio

A diverse collection of hardware-centric projects spanning sensor integration, real-time monitoring, and low-level hardware communication using ESP32, Arduino, and FPGA.

---

## 🏗️ Project Categories

### 📊 [CPU & System Monitoring](./CPU_Dashboard)
Real-time hardware performance monitoring and web-based dashboards.
*   **[CPU Dashboard](./CPU_Dashboard)**: Web interface for ESP32-based hardware stats.

### 🏥 [Health & Medical](./Medical)
Integrated health tracking and medical assistance systems.
*   **[Medical Monitoring](./Medical)**: Sensor-based patient health tracking with Firebase integration.

### 🛡️ [Security & Infrastructure](./Attendance_Access)
Automated systems for security, access control, and transportation.
*   **[Attendance & Access](./Attendance_Access)**: RFID-based entry and logging.
*   **[Missile Defense System](./Missile_Defence_System)**: Automated target tracking and alerting simulation.
*   **[RFID Toll & Parking](./RFID_Toll_System)**: Infrastructure automation for vehicle management.

### 🌿 [Environment & Monitoring](./Plant_Monitoring_System)
Automation for environmental sensing and response.
*   **[Plant Monitoring](./Plant_Monitoring_System)**: Soil and moisture tracking.
*   **[Weather Station](./Weather_Station)**: Localized meteorology sensors.
*   **[Rain Alarm & Smoke Detector](./Rain_Alarm)**: Critical safety alerts.

### 🧩 [Hardware Simulations](./FPGA_Project)
Low-level hardware description and architectural simulations.
*   **[FPGA Interfacing](./FPGA_Project)**: Verilog-based hardware design and serial communication.
*   **[SRAM Simulator](./SRAM_Simulator)**: Memory architecture simulation.

---

## 🛠️ Global Hardware Stack
- **Controllers**: ESP32, Arduino Uno/Nano, Tang Nano 9K (FPGA).
- **Sensors**: DHT11/22, RFID-RC522, PIR, Moisture, Ultrasonic.
- **Connectivity**: Wi-Fi (ESP32), Firebase, Serial (UART), HTTP.
- **Libraries**: All required Arduino/C++ libraries are located in the **[libraries/](./libraries)** directory.
