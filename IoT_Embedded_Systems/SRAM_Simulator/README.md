# 🧩 SRAM Simulator (ESP32)

A hardware-level simulation of Static Random Access Memory (SRAM) architectures using ESP32/Arduino.

## 🌟 Features
- **Memory Addressing Logic**: Simulates read/write operations with address decoding.
- **Bit-Level Manipulation**: Real-time visualization of memory states on the serial monitor or connected LEDs.
- **Performance Testing**: Measure access times and latency in various configurations.
