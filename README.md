## 🚀 Getting Started

Each project folder contains its own specific instructions for building and running. Generally:
1. **Java Projects**: Require JDK 17+ and Maven. Run using `mvn clean install`.
2. **IoT Projects**: Use the Arduino IDE or PlatformIO for uploading `.ino` files.
3. **Web Projects**: Open `index.html` in a browser or follow specific backend setup steps in their respective folders.
